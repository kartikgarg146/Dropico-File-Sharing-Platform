# Dropico – File Sharing Platform

Dropico is a PHP-based secure file-sharing system with:
- User login/register
- File uploads
- Shareable unique download links
- Download counter
- Modern UI & responsive design

### 🛠️ Tech Stack
- PHP 8+
- MySQL
- HTML / CSS
- XAMPP / WAMP (Local environment)

### 📂 Features
- Secure user authentication
- Upload any allowed file type (PDF, DOCX, PNG, JPG, TXT)
- Auto-generated unique token-based share links
- Download statistics per file
- Clean dashboard UI with modern gradients
- Owner-only file deletion
